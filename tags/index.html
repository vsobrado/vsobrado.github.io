Voicemail MongoDB query for OpenUC 16.04+ Mailbox report

  • August 29, 2018
  • ~1 min read

Once upon a time, the best way to check on the number of stored voicemails was by using the built in Java tool.</p>

As of version 16.04 which comes with an upgraded version of MongoDB, we can simply run this query instead of checking individual mailboxes with a bash script.

db.fs.files.aggregate([
    {"$group":
        {_id:"$metadata.user", 
             "Inbox":{
                    "$sum":{
                        "$cond":{if :{"$eq":["$metadata.folder", "inbox"]}, then: 1, else: 0}
                         }
                   },
             "Saved":{
                    "$sum":{
                        "$cond":{if :{"$eq":["$metadata.folder", "saved"]}, then: 1, else: 0}
                         }
                    },
             "Conference":{
                    "$sum":{
                        "$cond":{if :{"$eq":["$metadata.folder", "conference"]}, then: 1, else: 0}
                         }
                   },
             "Deleted":{
                    "$sum":{
                        "$cond":{if :{"$eq":["$metadata.folder", "deleted"]}, then: 1, else: 0}
                         }
                   }
            }
    }
    ])
